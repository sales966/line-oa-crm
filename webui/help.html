使用說明

LINE OA 客戶進度中樞

📖 這是什麼系統

本系統把散落在 LINE 官方帳號(chat.line.biz)的客戶對話,自動同步進一個共用後台, 讓跟單、設計、客服、管理各角色在同一個地方掌握每位客戶的進度、對話、檔案與內部討論。

  • 對話自動同步:同仁在 LINE 與客戶的對話,由瀏覽器插件自動搬進後台,不必手動複製。
  • AI 總結:系統自動整理每位客戶的重點(產品、數量、規格、預算)與後續待辦,並可人工修改與批注。
  • 進度紅綠燈:五個階段的任務清單一目了然,知道案子走到哪、還缺什麼。
  • 檔案牆:客戶在 LINE 傳的檔案與同仁上傳的檔案集中收納,可預覽、下載。
  • 內部討論:團隊在客戶頁內部留言、@ 提及同事或檔案,客戶完全看不到。

➕ 同仁如何用插件加客戶同步

後台的資料來自一個獨立的 Chrome 瀏覽器插件(與公司報價助手互不影響)。 每位需要同步客戶的同仁,在自己的電腦裝一次即可。

  1. 安裝插件 用 Chrome 開啟「擴充功能 → 管理擴充功能」,開啟右上角「開發人員模式」, 點「載入未封裝項目」,選擇 IT 提供的 extension-sync 資料夾(或解壓提供的 zip)。
  2. 填後端網址與 Token 在插件的「選項(Options)」頁,填入後端網址(公司內網,預設 http://localhost:4680) 與 IT 給的同步 Token。Token 用來驗證身分,請勿外流。
  3. 在 chat.line.biz 加入同步 用瀏覽器登入 chat.line.biz,打開要追蹤的客戶聊天室, 點插件視窗(popup)的「➕ 將此客戶加入同步」,或聊天室頁面左下角的「⟳ 同步」小按鈕。 加入後,系統會在每天 10:00–22:00 每小時自動增量同步,有新訊息時也會即時抓取。
  4. 需要完整歷史時按「建檔」 後台客戶列表或詳情頁有「🔄 建檔」按鈕,按下後插件會把該客戶的完整歷史對話翻到底並下載所有檔案, 完成後自動產生 AI 總結。狀態徽章會顯示「未建檔 / 建檔中 / 已建檔 / 失敗」。
⚠️ 插件只會同步你「主動挑選」的客戶(白名單),不會掃描整個聊天列表; 資料只送到公司後端,不會傳給任何第三方。

🖥 後台怎麼用

從「客戶列表」點任一客戶進入「客戶詳情」頁,詳情頁集中了以下功能:

🤖 AI 總結(可編輯・批注)
系統自動整理對話重點與後續待辦。可直接「修改」總結內容(會標記「已由 XXX 修改」並可還原),或加「批注」補充說明而不改原文。按「重新生成總結」會依最新對話重算。
📊 進度五階段紅綠燈
橫向步驟條顯示目前階段;展開每個階段可看到任務清單,🟢 已完成 / 🔴 未完成。點燈可手動切換(切換後標「手動」),系統的 AI 判定不會覆蓋手動設定。已回簽/已打樣可填天數、已出廠可填物流單號。
💬 對話時間軸
1:1 還原 LINE 對話:客戶訊息靠左灰底、我方靠右綠底,圖片內嵌縮圖、貼圖、檔案卡片、日期分隔一應俱全。可往上捲載入更早的訊息。
🗂 檔案牆(可上傳)
客戶在 LINE 傳的檔案與同仁上傳的檔案集中一處,可依「圖片 / PDF / 設計檔 / 文件 / 其他」分類篩選。圖片點開全屏預覽、PDF 開新分頁檢視。按「⬆ 上傳檔案」可自行上傳。
💬 內部討論 + @ 提及
團隊在客戶頁內部留言討論,客戶看不到;AI 會把這裡的內容當成背景記憶納入總結。輸入 @ 可提及同事(對方會收到鈴鐺通知)或檔案。也可在討論框用 📎 直接上傳檔案並分享。
🔔 通知鈴鐺
右上角鈴鐺顯示你被 @ 提及的未讀數,點開可看清單,點項目直接跳到該客戶的討論區並標記已讀。
📜 變更紀錄
所有關鍵操作(階段點燈、總結編輯/批注、檔案上傳等)都會留下「誰、何時、做了什麼」的紀錄,可展開查閱。
👤 帳號與權限
右上角可修改密碼。「管理」角色多一個「使用者管理」入口,可新增/停用帳號、重設密碼。

📊 進度五階段含義

案件固定依下列五個階段推進(流失為旁支狀態):

洽談
洽談
初步接觸:了解需求(盒型/尺寸/數量)、取得聯絡資訊、了解預算、出報價單。
▼
已回簽
已回簽 = 案件成立
客戶回簽報價單,案件正式成立。收打樣訂金、安排打樣、寄出樣品、客戶確認樣品。可設定「打樣天數」估算完成日。
▼
已打樣
已打樣 = 確定生產
客戶確認樣品後決定量產。收生產訂金/確認下單、安排大貨製作、確認交期。可設定「生產天數」估算完成日。
▼
已出廠
已出廠 = 物流
貨已出廠,進入物流階段。安排物流、填國際物流資料(物流商/單號/報關)、通知客戶出貨。
▼
已交付
已交付
客戶簽收、尾款結清、結案。
💡 系統會依對話內容用 AI 自動判斷目前階段,但你隨時可手動點燈調整或「鎖定」階段; 手動設定優先,AI 不會覆蓋。

🏷 客戶列表新工具(標籤・搜尋・匯出)

客戶列表最上方的工具列與每一列右側,新增了幾個提升找人、分類與整理效率的工具。

🏷 客戶標籤

  • 在哪:客戶列表每一列都有「🏷 標籤」鈕;工具列有「依標籤篩選」的選項。
  • 怎麼用:點該列的「🏷 標籤」鈕,勾選要貼的標籤(例如 急件 / VIP / 負責業務),一位客戶可同時貼多個。要只看某類客戶時,用工具列的標籤篩選。
  • 管理員:在「管理標籤」頁可新增標籤、改顏色、改名、刪除標籤定義。刪除標籤會從所有客戶身上一併移除。
  • 注意:標籤是全公司共用的,任何人貼上或管理員改動,大家看到的都一樣,請沿用團隊共識的命名。

🔍 全站搜尋

  • 在哪:客戶列表工具列的搜尋框。
  • 怎麼用:輸入關鍵字後,不只比對客戶名稱,還會搜「對話內容」與「AI 總結內容」;點搜尋結果可直接進入該客戶頁。
  • 注意:適合「記得跟某客戶聊過某個產品或規格,但忘了是哪一位」的情況,用內容關鍵字就能撈回來。

⬇ 匯出 CSV

  • 在哪:客戶列表工具列的「匯出 CSV」鈕。
  • 怎麼用:一鍵下載全部客戶清單(含階段、死線、總結、標籤等欄位),用 Excel 直接打開即可。
  • 注意:檔案已處理好中文編碼,用 Excel 開不會亂碼;適合整理報表或離線盤點。

▦ 看板視圖

  • 在哪:從上方導覽列點「看板」進入。
  • 版面:共 6 欄,對應 洽談 / 已回簽 / 已打樣 / 已出廠 / 已交付 五個階段,再加一欄「流失」,一眼看清每個階段各卡著哪些客戶。
  • 怎麼用:用滑鼠把客戶卡片拖曳到別欄,就等於人工釘選該客戶的階段(會覆蓋 AI 的自動判斷)。想恢復自動判定時,點卡片上的「↺ 自動」即可。
  • 注意:拖曳釘選=手動鎖定階段,之後 AI 不會再自動調整;若希望系統重新依對話判斷,記得按「↺ 自動」解除釘選。

🖨 客戶頁新功能(串流總結・列印)

⚡ 總結串流生成

  • 在哪:客戶頁按「重新生成總結」時。
  • 怎麼用:文字會即時一句句浮現(像打字機一樣),可以邊看邊等,不必盯著轉圈苦等整段跑完。
  • 注意:若你的瀏覽器不支援串流,系統會自動退回原本「整段一次出現」的方式,最後結果一樣,不影響使用。

🖨 列印 / PDF

  • 在哪:客戶頁的「列印 / PDF」鈕。
  • 怎麼用:點下後開啟列印視窗,可直接印出該客戶的總結、進度與死線;在印表機選單選「另存為 PDF」即可存成檔案。
  • 注意:很適合開會、交接或給主管過目時,做成紙本或 PDF 帶著走。

⏰ 待辦提醒增強

  • 在哪:首頁(客戶列表)上方的待辦提醒區。
  • 新增內容:除了原本的「死線將到」「打樣到期」「生產到期」,再新增兩種提醒——待建檔(還沒建檔的客戶)與尚未生成總結的客戶。
  • 怎麼用:點提醒項目可直接跳到對應客戶處理。看到「待建檔」就去按「🔄 建檔」補歷史;看到「尚未生成總結」就進客戶頁按「重新生成總結」。
  • 注意:把這幾項清空,代表每位客戶都已建檔且有最新總結,團隊掌握度最完整。

🛡 管理員專用:備份與審計日誌

以下兩項都在「系統健康」頁,僅「管理」角色可見。

💾 資料庫備份

  • 在哪:管理員的「系統健康」頁。
  • 自動:系統每 6 小時自動備份一次整個資料庫,並保留最近 30 份,更舊的會自動清除,不用擔心塞爆硬碟。
  • 怎麼用:在頁面上可看到備份清單、按「立即備份」手動備份一份,或按「下載」把備份檔存到自己電腦保管。
  • 注意:遇到重要節點(例如剛大量建檔完)建議手動備份一份並下載留底,萬一機器出狀況可還原。

📜 審計日誌

  • 在哪:管理員的「系統健康」頁。
  • 內容:全站變更紀錄——誰、在何時、做了什麼(登入、編輯總結、階段點燈、貼標籤等),便於稽核與追查。
  • 怎麼用:可依「使用者」或「動作類型」篩選,快速定位某項變更是哪位同仁在什麼時候做的。
  • 注意:僅管理角色可查閱;這是釐清「誰改了什麼」時的第一手依據。

❓ 常見問題排除

Q正在同步時可以操作嗎?會不會出錯?
同步進行中(尤其是「建檔」翻歷史時)請不要重複按同步或關掉 chat.line.biz 分頁。 系統有防重疊機制,重複觸發會顯示「同步進行中」;耐心等徽章變成「已建檔」即可。
Q插件顯示 Token 錯誤 / 一直同步失敗?
到插件「選項」頁確認後端網址與 Token 是否正確(Token 由 IT 提供,不能有多餘空白)。 若近期更換過 Token,請重新填入並儲存。
Q提示「LINE 登入已失效」?
代表 chat.line.biz 的登入過期了。請重新開啟 chat.line.biz 登入, 之後插件就能繼續同步。這與後台系統的帳號登入是兩回事。
Q後台打不開 / 顯示「無法連線到後端服務」?
確認後端程式已啟動且連接埠為 4680,並確認你的電腦與後端在同一個內網。 若仍不行,請聯繫 IT 或用下方「問題回報」提交。
Q被登出了,或按返回鍵跳到登入頁?
登入 session 有效期為 7 天,過期需重新登入。重新登入後系統會帶你回原本的頁面。
Q發現系統的問題想回報?
點右上角使用者選單的「🐞 問題回報」,填標題與說明送出。在客戶頁回報會自動關聯該客戶,方便管理員排查。
